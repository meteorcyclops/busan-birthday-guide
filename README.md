# 釜山生日旅行行程 / Busan Birthday Trip Itinerary

**日期 / Dates:** 2026/05/15 - 2026/05/19  
**旅程風格 / Style:** 文青散步、海景咖啡、好吃的、生日感  
**航班建議 / Flight:** 釜山航空 Busan Air，TPE → PUS，2026/05/15 - 2026/05/19

---

## Day 1 - 5/15（五）抵達日 / Arrival + Nampo Evening

### 流程 / Plan
傍晚抵達釜山後，先到飯店 check-in。第一晚不排太滿，建議直接去 **南浦洞 Nampo-dong** 一帶散步，先感受釜山街頭氣氛。可以先逛 **BIFF 廣場 BIFF Square**，再視體力決定要不要走到 **札嘎其市場 Jagalchi Market** 附近。晚餐以烤肉或豬肉湯飯為主，讓旅程舒服開始。

### 地點 / Places
- **南浦洞 Nampo-dong**  
  <https://www.google.com/maps/place/Nampo-dong,+Busan/@35.0989,129.0302,16z>
- **BIFF 廣場 BIFF Square**  
  <https://www.google.com/maps/place/BIFF+Square/@35.0988,129.0285,17z>
- **札嘎其市場 Jagalchi Market**  
  <https://www.google.com/maps/place/Jagalchi+Market/@35.0975,129.0306,17z>

### 晚餐 / Dinner
- **味贊王鹽烤肉 南浦店 / Matjanddeul Wang Sogeum Gui Nampo**

## Day 2 - 5/16（六）海景文青日 / Haeundae + Cheongsapo Seaside Day

### 流程 / Plan
早餐後前往 **海雲台 Haeundae**，先在海邊附近吃一頓漂亮的早午餐，再去 **海雲台海水浴場 Haeundae Beach** 散步。之後搭乘 **海雲台藍線公園 Haeundae Blueline Park** 的海岸列車或膠囊列車，慢慢移動到 **青沙浦 Cheongsapo**。中午可安排海鮮或烤貝，下午找海景咖啡店休息，晚上去 **The Bay 101** 看夜景。

## Day 3 - 5/17（日）文青散步日 / Huinnyeoul + Jeonpo Cafe Day

## Day 4 - 5/18（一）生日主日 / Birthday Main Day at Gwangalli

這天整體節奏放慢，重點放在生日氛圍。上午前往 **廣安里海水浴場 Gwangalli Beach** 看海與 **廣安大橋 Gwangan Bridge** 景色。中午可在 **民樂 The Market Millac the Market** 周邊吃飯，再找一間海景咖啡店坐坐。下午保留一點留白，像是散步、拍照、送禮物。晚上安排整趟最好的晚餐，之後在海邊散步收尾。

## Day 5 - 5/19（二）回程日 / Departure Day

### 流程 / Plan
最後一天以輕鬆為主，在飯店附近吃早餐後直接前往機場。不要再塞景點，讓整趟旅行舒服結束。

### 早餐 / Breakfast
如果住 **西面 Seomyeon**：  
- **西面早餐咖啡店 / Breakfast Cafe Seomyeon**  
  <https://www.google.com/maps/search/?api=1&query=Breakfast+Cafe+Seomyeon+Busan>

如果住 **海雲台 Haeundae**：  
- **海雲台早餐咖啡店 / Breakfast Cafe Haeundae**  
  <https://www.google.com/maps/search/?api=1&query=Breakfast+Cafe+Haeundae+Busan>
